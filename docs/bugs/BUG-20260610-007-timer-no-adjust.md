# Timer 模式无法调整倒计时时长

- **Bug ID**：BUG-20260610-007
- **严重等级**：P2-一般
- **发现日期**：2026-06-10
- **修复日期**：—

## 现象描述
单键模式下只能长按开始/暂停/重置倒计时，无法增减倒计时分钟数。当前固定为 `TIMER_DEFAULT_MINUTES`（5 分钟）。

## 复现步骤
1. 进入 Timer 模式（STOPPED 状态）
2. 想从 5 分钟改成 15 分钟
3. 无任何操作可调整时长

## 根因分析
单键适配后移除了 UP/DOWN 按键逻辑。单键在 STOPPED 状态的长按被"启动倒计时"占用，无多余操作可分配。

## 修复方案
- 双击 → 进入时长调整模式（闪烁显示分钟数）
- 长按 → 调整模式内每分钟 +1，循环 1→5→10→15→30→60→1
- 三击或超时 3 秒无操作 → 退出调整模式

## 影响文件
- `firmware/App/modes/timer_mode.c`
- `firmware/Core/Inc/app_config.h`（TIMER_DEFAULT_MINUTES）

## 验证方式
双击进入调整模式 → 长按递增 → 超时退出 → 启动倒计时验证时长正确

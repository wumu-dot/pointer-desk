# Settings 模式只读，无法修改设置项

- **Bug ID**：BUG-20260610-006
- **严重等级**：P2-一般
- **发现日期**：2026-06-10
- **修复日期**：—

## 现象描述
进入 Settings 模式后只能查看各个设置项，无法修改值（如亮度增减、单位切换）。

## 复现步骤
1. 短按切到 Settings 模式
2. 长按 = 退出保存（当前唯一操作）
3. 无按键可修改设置项的值

## 根因分析
单键操作限制：短按由 mode_manager 接管（切模式），长按 = 退出。缺少方向键或确认键来修改设置值。

## 修复方案
简化 Settings 交互：每次进入 Settings 只显示一个设置项，长按 = 修改当前项的值（循环切换），短按 = 下一个设置项。或者接受现状（设置通过编译时宏修改，Flash 仅持久化）。

## 影响文件
- `firmware/App/modes/settings_mode.c`

## 验证方式
进入 Settings 后能用单键修改亮度/温度单位等值

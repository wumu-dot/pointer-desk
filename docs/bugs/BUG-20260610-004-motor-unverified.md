# 步进电机 + A4988 未验证

- **Bug ID**：BUG-20260610-004
- **严重等级**：P1-严重
- **发现日期**：2026-06-10
- **修复日期**：2026-06-23

## 现象描述
指针引擎代码（`pointer_engine.c`）和 A4988 驱动（`a4988.c`）编译通过且逻辑完整，但因硬件未接无法物理验证。当前电机任务空转输出 PWM 但无实际负载。

## 复现步骤
1. 观察串口日志 `Pointer target set: 80.0 deg` / `Pointer reached target: 80.0 deg`
2. 无物理指针运动

## 根因分析
42 步进电机 + A4988 驱动板已购买，但缺少 12V 适配器供电。

## 修复方案
购买 12V 2A DC 适配器 + 100µF 电解电容（并在电机电源端滤波），接线：
- **PA0 = STEP**（TIM2_CH1 PWM）
- **PA1 = DIR**
- **PA2 = EN**（低有效）
- **PA3 = MS1/MS2/MS3**（三根跳线并接 = 16 细分）
- A4988 逻辑供电 3.3V 从面包板 +轨取
- A4988 VMOT 接 12V 适配器

## 影响文件
- `firmware/Drivers/BSP/motor/a4988.c`
- `firmware/Middleware/pointer/pointer_engine.c`

## 验证方式
接入后切模式，观察指针物理运动：温度模式指针指向当前温度刻度、倒计时模式指针随倒计时转动

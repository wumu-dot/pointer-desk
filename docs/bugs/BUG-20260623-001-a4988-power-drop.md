# A4988 旧模块 12V 供电拉跨，电机不转

- **Bug ID**：BUG-20260623-001
- **严重等级**：P0-致命
- **发现日期**：2026-06-23
- **修复日期**：2026-06-23

## 现象描述
A4988 代码逻辑正确、EN/STEP/DIR 信号正常，但电机完全不转。适配器 12V 接蓝色端子 VIN 正常输出，但一插上电机，VIN 立即从 12V 掉至 0.35V。

## 复现步骤
1. 适配器 12V 接入蓝色端子 VIN
2. 万用表量 VIN = 12V（正常）
3. 插上 42 步进电机
4. VIN 瞬间掉到 0.35V，电机不转，A4988 模块灯灭

## 根因分析
旧 A4988 模块内部存在质量问题，插电机后瞬时电流超过适配器 1.5A 保护阈值，适配器过载拉低电压。万用表测电机线圈 φ：同相 1~5Ω，无短路过。

Vref 从 0V 调到 0.9V 均无效——只要电机接上，VIN 就掉 0.35V，A4988 芯片无电工作。

## 修复方案
换新 A4988 驱动模块。新模块同接线、Vref=0.8V、12V 适配器不降，电机正常锁住、能转。

## 影响文件
- `firmware/Drivers/BSP/motor/a4988.c`
- `firmware/Core/Src/freertos.c`

## 验证方式
换新模块 → Vref=0.8V → 上电 → 电机锁住（手拧不动）→ 自检 CW/CCW 慢转正常 → VIN 稳定 12V 不掉

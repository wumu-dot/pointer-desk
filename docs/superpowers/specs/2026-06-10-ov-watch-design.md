# OV-Watch 桌面智能摆件 — 设计文档

> 基于 STM32F407ZGT6 + 物理指针 + TFT 的桌面混合智能摆件
>
> 日期：2026-06-10

---

## 1. 项目概述

本项目参考 B站UP主"油炸鸡开源硬件"(GitHub: No-Chicken) 的开源项目 OV-Watch，制作一个**简化版桌面智能摆件**。核心特色是 **TFT屏幕 + 42步进电机驱动物理指针** 的混合显示交互方式。

### 1.1 硬件清单

| 组件 | 型号 | 接口 | 备注 |
|------|------|------|------|
| 主控 | STM32F407ZGT6 | — | Cortex-M4F, 168MHz, 1MB Flash, 192KB SRAM, 144pin |
| 屏幕 | 1.8寸 TFT (ST7735S) | SPI1 | 128×160, 彩色 |
| 存储 | W25Q64 | SPI2 | 8MB, LittleFS |
| 电机 | 42步进电机 (NEMA17) | A4988 驱动 | 200步/圈, 16细分=3200微步/圈 |
| 电机驱动 | A4988 | TIM2 PWM | STEP/DIR 控制, 12V供电 |
| 输入 | 四角开关 | GPIO | 5方向: 上/下/左/右/按下 |
| 温度 | 内部ADC (初期) + 预留SHT30 | ADC1 / I2C2 | 扩展排针引出 |
| 时钟 | STM32内置RTC | VBAT | CR1220纽扣电池备份 |
| 供电 | 12V DC适配器 → AMS1117-3.3V | — | 桌面USB/适配器供电 |

### 1.2 开发环境

- **IDE**: STM32CubeIDE
- **HAL**: STM32Cube HAL 库
- **RTOS**: FreeRTOS (CMSIS-OS v2)
- **文件系统**: LittleFS

### 1.3 搭建方式

**阶段1 — 面包板原型**：
- STM32F407开发板 + 各模块（TFT/A4988/W25Q64/按键）用杜邦线连接
- SPI 频率降至 8-9MHz 保证信号稳定
- SCK/MOSI 之间夹一根 GND 线减少串扰
- A4988 电源端加 100μF 电解电容，远离 TFT 模块摆放

**阶段2 — 定制PCB整合**（固件验证完成后）：
- 全部元件焊接在一块 PCB 上
- TFT 用 FPC 排线插座，A4988 用排母（可插拔）
- 表盘外形安装物理指针

```
│                            │      │                   │

│  电机任务 = 接收目标角度，50ms中断微步逼近            │
│  按键任务 = GPIO轮询 + 去抖(20ms) + 事件分发          │

```

### 2.1 任务详解

| 任务 | 栈 | 优先级 | 周期 | 职责 |
|------|-----|--------|------|------|
| 按键任务 | 512B | 高 | 10ms轮询 | GPIO读取 → 去抖 → 识别短按/长按/方向 → 发送事件到后台队列 |
| 后台任务 | 2KB | 中 | 事件驱动 | 状态机管理、模式切换、传感器读取、RTC维护、角度计算、发送渲染指令 |
| 显示任务 | 2KB | 中 | 30fps | 接收渲染指令 → 脏矩形局部刷新TFT，不参与计算 |
| 电机任务 | 1KB | 中 | 50ms中断 | 接收目标角度 → 线性插值逼近 → TIM2输出STEP脉冲 → 到位后发事件 |

### 2.2 任务间通信

```
按键 → 后台:  osMessageQueue (按键事件结构体)
后台 → 显示:  osMessageQueue (渲染指令: 区域+内容)
后台 → 电机:  osMessageQueue (目标角度+速度档位)
电机 → 后台:  osEventFlags (到位标志位)
```

### 2.3 关键优化

- **TFT局部刷新**：显示任务维护脏矩形列表，只刷新变化的像素区域
- **电机硬件驱动**：TIM2 CH1输出PWM到A4988 STEP脚，硬件产生脉冲不占CPU
- **SPI隔离**：TFT用SPI1，Flash用SPI2，物理上不冲突
- **定时器资源**：TIM2→A4988 STEP, TIM3→TFT背光PWM, TIM4→50ms电机微步, TIM5→按键轮询

## 3. 模式功能详述

### 3.1 时钟模式（默认启动）

| 项目 | 说明 |
|------|------|
| **TFT显示** | 时间 HH:MM:SS（大字体）、日期 YYYY-MM-DD、星期（中文）、AM/PM图标（太阳/月亮）、底部24小时浓缩时间条 |
| **指针行为** | 平滑扫过小时刻度（12等分表盘），50ms微步中断驱动，每微步0.1125° |
| **12/24h切换** | 上下键切换，24h制时指针转一整圈为24小时，TFT图标同步切换 |
| **断电保护** | RTC 接 CR1220 纽扣电池（VBAT），首次上电自动进入时间设置 |
| **AM/PM区分** | 12h制：TFT右上角显示太阳/月亮图标；屏幕底部24h时间条用白点标记当前时刻 |

**指针算法**：
```
目标角度 = (小时 % 12) × 30° + 分钟 × 0.5°
// 后台任务每秒更新目标角度
// 电机任务每50ms执行: current += (target - current) / 20
// 20步=1秒平滑到达，视觉上无跳跃感
```

### 3.2 温度计模式

| 项目 | 说明 |
|------|------|
| **TFT显示** | 温度数值（大字体）、温度趋势箭头、弧形虚拟刻度盘（-10~50°C，300°弧）、传感器精度来源标记 |
| **指针行为** | 指向温度刻度盘对应位置，5秒刷新一次 |
| **℃/℉切换** | 上下键切换单位，指针和刻度盘同步更新 |
| **传感器** | 初期：STM32内部ADC温度（误差±5°C，标记"仅供参考"）。预留I2C2 SHT30接口（精度±0.3°C），插上自动切换 |

**指针映射（线性）**：
```
角度 = (温度 - (-10)) / 60 × 300°
// 温度 20°C 对应角度 = 30/60×300 = 150°
// A4988 3200微步/圈 × (300°/360°) / 60°C = 44微步/°C
// 每微步 ≈ 0.022°C，远超显示精度需求
```

### 3.3 计时器模式

| 项目 | 说明 |
|------|------|
| **TFT显示** | 倒计时/正计时数字 MM:SS、状态标签（运行/暂停/停止）、进度环（弧形条） |
| **指针行为** | 倒计时：指针从满刻度逐渐归零（一圈=1小时）。最后10秒每秒跳一格(6°)，TFT数字变大变红 |
| **操作逻辑** | 短按上键+1分钟、长按(>0.5s)上键+10分钟；下键同理减；中键开始/暂停；左键复位 |
| **结束提示** | 电机三次短震（哒-哒-哒，占空比50%，间隔200ms），TFT闪烁"时间到！"5秒，自动返回时钟模式 |

**倒计时指针算法**：
```
剩余比例 = 剩余时间 / 总时间
目标角度 = 剩余比例 × 360°
// 正常阶段：50ms微步平滑移动
// 最后10秒：每秒跳6°（大步），配合电机震动
```

### 3.4 设置菜单

| 项目 | 说明 |
|------|------|
| **分页结构** | 第1页：亮度、日期时间、温度单位。第2页：计时默认时长、恢复出厂、关于 |
| **TFT显示** | 当前页菜单列表，当前项高亮反色显示 |
| **指针行为** | 指向当前页码刻度（第N页刻度=N），辅助定位 |
| **操作逻辑** | 上下键移动光标，中键确认进入子项，左键返回上一级 |
| **亮度调节** | TIM3 PWM → TFT背光 (0~100%, 10档) |
| **恢复出厂** | 擦除LittleFS配置区，保留固件，需二次确认防误触 |
| **关于** | 固件版本号、编译日期、开源地址 |

## 5. 引脚分配

### 5.1 SPI1 — TFT (ST7735S)

| 引脚 | 功能 | 说明 |
|------|------|------|
| PA5 | SCK | 时钟, 8-9MHz |
| PA7 | MOSI | 主机输出 |
| PA4 | CS | 片选 (软件控制) |
| PC4 | DC | 数据/命令选择 |
| PC5 | RST | 复位 |
| PB0 | BL | 背光 (TIM3 CH3 PWM) |

### 5.2 SPI2 — Flash (W25Q64)

| 引脚 | 功能 | 说明 |
|------|------|------|
| PB13 | SCK | 时钟 |
| PB15 | MOSI | 主机输出 |
| PB14 | MISO | 从机输入 |
| PB12 | CS | 片选 (软件控制) |

### 5.3 步进电机 (A4988)

| 引脚 | 功能 | 说明 |
|------|------|------|
| PA0 | STEP | TIM2 CH1 PWM 脉冲 |

| PA2 | EN | 使能 (低有效) |
| PA3 | MS1/MS2/MS3 | 细分设置 (16细分: MS1=1, MS2=1, MS3=1) |

### 5.4 四角开关

## 6. Flash 存储规划 (W25Q64, 8MB)

```
┌────────────────────────────┐
│ 0x000000 - 0x0FFFFF (1MB)  │  LittleFS 配置区
│                             │  - 系统设置 (亮度/单位/默认时长)
│                             │  - 用户偏好
├────────────────────────────┤
│ 0x100000 - 0x3FFFFF (3MB)  │  字体资源
│                             │  - 中文字库 (GB2312 常用字)
│                             │  - ASCII 多字号字体
├────────────────────────────┤
│ 0x400000 - 0x7FFFFF (4MB)  │  图片资源
│                             │  - 表盘素材
│                             │  - 图标集
│                             │  - 启动Logo
└────────────────────────────┘
```

- LittleFS 提供磨损均衡和掉电保护
- 字体和图片资源通过 SPI Flash 烧录工具预写入
- 配置区单独挂载 LittleFS 分区，恢复出厂时只擦除这个区域
